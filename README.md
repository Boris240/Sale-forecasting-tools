# Sale-forecasting-tools

Cette application web a été développée pour réaliser des prévisions de ventes sur le long terme (10 ans) et simuler différents scénarios selon des critères de validation.

Note importante :
Cette version du projet est nettoyée pour respecter la confidentialité des données et des systèmes de l’entreprise.
Tous les éléments sensibles ont été volontairement retirés (chemins, fichiers de données, configurations).
Le code n’est donc pas directement exécutable, mais il permet de visualiser la structure, la logique de modélisation et l’architecture de l’application.

# Fonctionnalités

- Prévision automatique des ventes pour plusieurs pays via un modèle State Space.

- Interface web pour simuler différents scénarios de ventes selon des critères de validation.

- Visualisation des résultats à travers des graphiques et tableaux interactifs.

- Déploiement prévu sur Microsoft Azure (version originale).

# Technologies utilisées

- R et Shiny pour le développement web.

- Modèles dynamique Espace Etat pour les prévisions.

- Packages R : forecast, shiny, dplyr, KFAS et bien d'autres...

# Objectif du projet

L’objectif principal est de montrer :

- La logique de prévision à long terme

- L’architecture d’une application web Shiny professionnelle

- La mise en place d’un outil de simulation et de validation

Cette version permet de présenter le projet dans un portfolio ou sur GitHub sans exposer de données réelles ou confidentielles.

# Résultats

- L’application originale permettait de prévoir les ventes par pays sur 10 ans.

- Les utilisateurs pouvaient simuler différents scénarios et valider les prévisions avec des critères définis.

- Déploiement initial sur Azure pour usage interne.

# Remarques pour le lecteur

Le code est fourni à titre démonstratif uniquement.

Pour reproduire l’exécution complète, il serait nécessaire d’utiliser les données réelles et la configuration originale, non disponibles pour des raisons de confidentialité.
